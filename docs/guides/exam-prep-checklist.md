# Exam Preparation Checklist

Use this checklist in the final days before a COA exam or assessment.

## Concepts To Recheck

- I can explain the purpose of each of the 5 pipeline stages.
- I can identify RAW, WAR, WAW, structural, and control hazards.
- I know when forwarding solves a dependency and when a stall is still required.
- I can explain the difference between latency and throughput.
- I can reason about branch penalties and CPI impact.

## Problem Types To Practice

- pipeline timing tables
- load-use hazard questions
- branch penalty calculations
- AMAT and cache address breakdown problems
- CPI and speedup comparisons

## Tool-Based Revision

- Use `tools/pipeline-simulator.jsx` for instruction-sequence tracing.
- Use `tools/formula-brain.jsx` to rehearse critical formulas.
- Revisit Lesson 3 if hazard questions still feel shaky.

## Last-Minute Review Plan

### 60 Minutes Available

1. Review critical formulas.
2. Solve one hazard problem.
3. Revise branch penalties and CPI.

### Half-Day Available

1. Read the lesson summaries.
2. Run simulator practice on 3-5 instruction sequences.
3. Review formula flashcards.
4. Write down your recurring mistakes once.

### One Full Day Available

1. Follow `docs/guides/study-guide.md`.
2. Focus on weak areas only.
3. End with timed mixed practice instead of passive reading.

## Final Confidence Check

- I can solve a basic pipeline table without notes.
- I can tell whether a stall is required and why.
- I can choose the correct formula before substituting values.
- I can explain mistakes, not just memorize answers.
