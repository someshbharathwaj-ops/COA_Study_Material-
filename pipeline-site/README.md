# COA Study Platform

This React/Vite app is the interactive frontend for the COA study repository.

## Available Commands

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Source Files

- `src/App.jsx`: main learning interface
- `src/FormulaBrain.jsx`: embedded formula study module
- `src/main.jsx`: Vite entrypoint
- `src/base.css`: minimal global reset

## Relationship To The Repository

This app complements the standalone study assets in:

- `../lessons/`
- `../tools/`
- `../docs/`
