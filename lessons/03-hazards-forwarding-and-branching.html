📐 MEMORY FORMULAS

Hit Rate = Hits / Total
Fraction of accesses found in cache
Miss Rate = 1 − Hit Rate
Fraction that must go to lower level
Index bits = log₂(N_blocks)
Bits needed to address cache sets
Miss Rate = Misses / Total
Direct computation from access log
